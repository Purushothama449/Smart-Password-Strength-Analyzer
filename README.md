# 🔐 Smart Password Strength Analyzer

##  Overview

Smart Password Strength Analyzer is a cybersecurity-focused web application developed using **Python** and **Streamlit**. It helps users evaluate password security by analyzing password strength, entropy, crack time, common password usage, sequential patterns, repeated characters, and cryptographic hashes.

This project was developed as part of a **Cyber Security Internship at Decode Labs**.

---

##  Features

###  Password Security Analysis

* Password Strength Scoring (0–100)
* Weak / Medium / Strong Classification
* Entropy Calculation
* Estimated Password Crack Time

###  Security Checks

* Common Password Detection
* Breach Password Detection
* Repeated Character Detection
* Sequential Pattern Detection
* Password Policy Validation

###  Password Generator

* Generate Secure Random Passwords
* Strong Password Recommendations

###  Cryptographic Hash Generator

* MD5 Hash
* SHA1 Hash
* SHA256 Hash
* SHA512 Hash

###  PDF Security Report

* Download Detailed Password Security Report
* Includes Score, Entropy, Crack Time, and Security Status

###  Cyber Security Dashboard

* Dark Hacker-Themed UI
* Interactive Gauge Meter
* Real-Time Password Analysis

##  Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Smart-Password-Strength-Analyzer.git
```

Navigate to the project folder:

```bash
cd Smart-Password-Strength-Analyzer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
